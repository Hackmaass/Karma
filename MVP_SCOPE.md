# KarmaOS - MVP Scope (Hackathon Ready)

## What to Build (The Core Loop)
1.  **The Dashboard UI:** The exact React/Tailwind frontend designed here. It must look premium and function flawlessly as a prototype.
2.  **The AI Insight Generator (Backend):** A simple Node.js/Express endpoint that takes a hardcoded JSON payload of "raw activity" and uses the Gemini API to generate the "Good morning" summary sentence and the 3 key metrics.
3.  **The AI Assistant (Chat):** A functional chat interface that uses Gemini to answer questions based on a hidden, hardcoded context document about the "team's current state".

## What to Simulate (Fake it)
1.  **Data Ingestion:** Do NOT build real OAuth integrations for Slack, Jira, or Google Calendar. This takes weeks. Instead, create a script that generates realistic JSON "metadata" (e.g., `{"event": "slack_message", "user": "Sarah", "timestamp": "10:04 AM"}`).
2.  **Historical Data:** Hardcode the trend lines and historical comparisons ("+12% vs last week").
3.  **User Authentication:** Simulate a logged-in state. Don't build a real login flow.

## What to Ignore (Do not build)
1.  **Settings & Configuration:** Ignore the ability to add/remove team members or configure integrations.
2.  **Real-time Updates:** Don't build WebSockets. The dashboard can just load a static state on refresh.
3.  **Complex Data Visualization:** Stick to the "Meaningful Statements" and simple CSS progress bars. Do not integrate D3 or Chart.js. They add visual noise and development time.

## The 3-Minute Demo Flow
1.  **0:00 - 0:30:** Show the Landing Page. Emphasize the philosophy: "Understand impact, not activity." Click "View Demo".
2.  **0:30 - 1:30:** The Dashboard loads. Point out the primary insight: "Good morning. Your team is focused, but engineering is blocked..." Explain that this was generated by Gemini analyzing simulated Slack/Jira metadata, not by a human writing a status update.
3.  **1:30 - 2:00:** Navigate to the "Team Insights" page. Show the Collaboration Flow and Bottlenecks. Explain how this replaces gut-feeling management.
4.  **2:00 - 3:00:** Open the AI Assistant. Click the suggested prompt: "Who is overloaded right now?" Show Gemini responding with a nuanced answer about Sarah's context-switching fatigue. End on the value proposition: proactive burnout prevention.
